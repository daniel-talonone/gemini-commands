# Custom Gemini CLI Commands

This repository contains custom commands for the Gemini CLI.

## Project Overview

This project explores semi-automated development workflows. It uses commands to manage task state for feature planning, implementation, and review.

## Available Command Groups

The following command groups are available:

### `/session:*` Commands

This group of commands manages a session-based workflow for feature development. It uses a "feature directory" to store task state across sessions.

For details on these commands, see:

*   **[Session Workflow README](./session/README.md)**
